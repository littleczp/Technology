# 上下文

<details>

<summary>上下文切换</summary>

保存当前执行进程一些寄存器的值，为即将恢复执行的进程恢复一些寄存器的值（通用寄存器、程序计数器、内核栈帧指针）

</details>



上下文切换的两种场景

{% tabs %}
{% tab title="时钟中断，系统时间的维护和环境的切换" %}
由cpu使用被切换进程的内核栈，隐式保存进程的寄存器
{% endtab %}

{% tab title=" 时分共享，进程调度和管理" %}
操作系统决定从进程A切换到进程B，内核寄存器被明确地保存到进程B的内存中
{% endtab %}
{% endtabs %}
