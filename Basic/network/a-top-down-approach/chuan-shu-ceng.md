# 传输层

<details>

<summary>流量控制与拥塞控制</summary>

|      | 目的        | desc                                                                                                                     |
| ---- | --------- | ------------------------------------------------------------------------------------------------------------------------ |
| 流量控制 | 端到端的控制    | <p>A通过网络给B发数据（对端），A发送的太快导致B没法接收(B缓冲窗口过小或者处理过慢)，那么就会有分组丢失</p><p>原理是通过滑动窗口的大小改变来实现（流量控制根本目的是防止分组丢失，它是构成TCP可靠性的一方面）</p>     |
| 拥塞控制 | 防止路由器接收不住 | <p>A与B之间的网络发生堵塞导致传输过慢或者丢包，来不及传输。防止过多的数据注入到网络中，这样可以使网络中的路由器或链路不至于过载</p><p>拥塞控制是一个全局性的过程，涉及到所有的主机、路由器，以及与降低网络性能有关的所有因素</p> |



</details>
