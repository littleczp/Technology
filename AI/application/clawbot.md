# Clawbot

不同于传统的 Web 端 AI，Clawdbot 采用了本地优先（Local-first）的架构。它运行在你的 Mac 或服务器上，通过你最熟悉的聊天软件作为交互入口，**直接操控你的本地文件、终端、甚至浏览器**。

|      | 传统 AI 应用     | Clawdbot 智能体   |
| ---- | ------------ | -------------- |
| 交互入口 | 网页 / APP     | 任意IM           |
| 持久化  | 仅存于云端session | 本地markdown长期存储 |
| 执行能力 | 只能给出建议       | 直接运行           |
| 隐私安全 | 流向第三方服务器     | 配置和数据都在本地存储    |
| 主动性  | 被动响应         | 主动执行（定时、监控）    |
